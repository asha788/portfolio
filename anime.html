Anime Website Design

A modern anime-themed website created with vibrant visuals, clean layout, and smooth UI elements for browsing shows, characters, and episodes.

Role: UI/UX Designer | Tools: Figma | Timeline: 1 week

Brief

The goal was to redesign an anime website with a modern look, bold visuals, and a clean structure. The design focuses on improving the user experience by making anime discovery simple, visually appealing, and enjoyable.

Screenshots

Wireframes

Problem

The original design looked outdated and lacked strong visual identity. Navigation was unclear, and browsing anime felt cluttered.

Solution

I redesigned the interface with a bold anime-inspired style, vibrant color palette, clean content layout, and easy navigation. The result is a more immersive and user-friendly anime browsing experience.
