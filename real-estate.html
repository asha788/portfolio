Real-Estate Website Redesign

A clean, modern redesign focused on improving property discovery, readability, and user trust through a structured layout and updated UI components.

Role: UI/UX Designer | Tools: Figma | Timeline: 1.5 weeks

Brief

The original website was visually outdated and difficult to navigate. I redesigned the experience to create a clean layout, a modern grid system for showcasing properties, and improved visual hierarchy to help users quickly browse listings and access key information.

Redesign Screens

Problems

  • Cluttered layout with poor spacing.
  • Property cards lacked structure and hierarchy.
  • Navigation was difficult for new visitors.

Goals

  • Improve visual clarity and consistency.
  • Create a modern property grid layout.
  • Highlight key actions like inquiry and filtering.

Solution

A clean redesign introducing a structured layout, strong typography, simplified navigation, and an updated visual system tailored for real-estate browsing.
